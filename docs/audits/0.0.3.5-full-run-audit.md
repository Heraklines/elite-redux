- **Ace**: vanilla trainers only (no ER override) — by design.
- **Elite**: 77 trainer battles, **77 distinct ER rosters** (was 20). Every
  trainer is now a different ER team.
- **Hell**: 96 trainer battles, **96 distinct ER rosters** (was 19).

trainer-TYPE's small pool once exhausted; it now falls back to the GLOBAL unused
tier-eligible pool (428+ trainers) before any repeat, so runs field dozens of
distinct trainers. Wave-fraction strength ordering still applies, so early waves
stay weak and E4/champion-tier teams still surface late.
