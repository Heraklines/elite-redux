# Elite Redux 0.0.3.2

Bug fixes, a faithfulness pass on a few moves, and a big move-info upgrade.

## ✨ New Features

- **Battle info screen during move selection.** The detailed in-battle info screen (the "Pokémon Stats" / Abilities / Moves / Field panels) is now reachable with the **info button** (C / the on-screen Stats button / your controller's info button) *while choosing a move*, not just from the command menu. Pressing it in the fight menu used to pop up PokeRogue's plain stat-arrow overlay instead — now it opens the proper screen.
- **Elaborate move details in the Summary.** The Pokémon Summary → Moves page now shows each move's full properties (power, accuracy, type/category, and its mechanics), cyclable for the highlighted move. Works with the 5th move slot.

## 🐛 Bug Fixes

- **PP Up / PP Max (and other move items) no longer freeze the game** when used on a Pokémon that has the 5th move slot.
- **Leech Blade** now heals **50% of the damage dealt** (the drain was missing — it healed nothing). The heal scales with damage actually dealt, so a finishing blow on a weakened target heals less.
- **Toxic Plunge** is now a proper **two-turn move** — it dives on the first turn and strikes (with its 20% poison chance) on the next — instead of hitting instantly like a jab.
- **Aftermath** Pokémon are no longer left **invincible** by a rare timing case (their explosion fizzling when they KO the last foe with their own move).
- **iOS data import** now works: a visible "Select File to Import" button appears so you can move saves between a computer and an iPhone/iPad. _(Ported from the upstream PokeRogue fix.)_

_Reminder: clearing your browser's "site data" will wipe local saves — please don't. Use **Manage Data → Export** to back up._
