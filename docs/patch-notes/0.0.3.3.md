# Elite Redux 0.0.3.3

Crash fixes, a new challenge reward system, and balance.

## ✨ New Features

- **Challenge Favour → Shiny odds.** Handicapping yourself now pays off: every challenge that makes the run *harder* grants **Favour** (Hardcore 8, Limited Support 6–10, Mono-type / Mono-gen 5, Limited Catch 4, Fresh Start / Inverse / Flip-Stats / Lower-cost 3). The more favour your run carries, the higher its shiny rate — **+0.5× per 5 favour, up to 3× (triple) shiny odds**. The challenge-select screen shows a live shiny readout as you toggle challenges. *(**Active Passives** is a convenience, not a handicap — passives are on by default — so it grants no favour.)*
- **Redux forms now appear in the wild** (a modest chance on enemy/wild spawns, all modes) instead of essentially never.

## ⚖️ Balance

- **Azelf / Mesprit / Uxie (Redux)** starter cost raised from 3 → **6**, and they now hatch from **Epic** eggs (was Rare).
- **Unown** no longer dominates egg hatches — its egg-pull weight is cut to ~1/10, so it's a rare novelty rather than the default pull.
- **Stone & trade evolutions are now level-up**, matching the ER dex — Vulpix, Eevee, Gloom, Kadabra/Machoke/Graveler/Haunter (trades), Magneton and ~80 others no longer need a stone or trade; they evolve by level (multi-option lines like Eevee/Gloom let you pick).
- **Challenge favour** for **Limited Support** now scales with its setting: no heal **6**, no shop **8**, both **10**.

## 🐛 Bug Fixes

- **Run History no longer freezes.** Opening a past run that contained a Pokémon with the **5th move slot** (the Move Slot Expander rogue item) crashed the run-history screen — the move grid only had four positions, so the fifth move read past the end of the list and threw, hanging the screen. The grid now has a fifth slot and is bounded safely, so runs with five-move Pokémon open normally.
- **Kinesis can no longer perma-flinch.** It's a +1-priority move that flinches; being repeatable, the AI could flinch-lock you every turn. It's now first-turn-only (like Fake Out), so it disrupts on switch-in but can't loop.
- **Counter-attack abilities no longer loop on self.** A Pokémon with a "retaliate when hit" ability (e.g. Mega Chimecho's Wind Chimes → Hyper Voice) would, after using a **self-targeted move like Cosmic Power**, count itself as its own attacker and counter *itself* — re-triggering endlessly ("hit itself repeatedly"). These abilities now only fire against an actual opponent's damaging move.
- **Challenge runs no longer black-screen.** Starting a challenge (e.g. a **Mono-type** run) and leaving the challenge-select screen blacked out the starter screen. Building the starter grid under a challenge crashed on certain species whose form data was incomplete (many Elite Redux customs / evolution targets), aborting the whole screen. The grid now handles those species safely, so challenge runs start normally. *(Tested on PC & mobile.)*
- **Looter** (and other heal-on-attack abilities) now show a proper message instead of raw text (`postAttackHeal`) when they restore HP.
- Removed an unintended **egg-gacha voucher exploit** (a leftover dev shortcut that granted 999 of every voucher).
- **No more early megas in Ace & Elite.** Elite Redux trainers could field a Mega-evolved Pokémon very early because a mon holding a Mega Stone always Mega-evolved. In **Ace** and **Elite**, mega evolution now waits until later waves so it's the late-game power spike it's meant to be. **Hell is unchanged** — early megas are part of its difficulty.
