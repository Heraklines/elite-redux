# Elite Redux 0.0.3.4

Bug fixes and egg-tier corrections.

## ⚖️ Balance

- **Thundurus and Silvally forms now hatch from Epic eggs.** Elite Redux ships their alternate forms (Thundurus Therian, every Silvally type form) as separate species, which were slipping into **Rare** eggs. They're now **Epic**, matching their base — no more legendary forms from Rare pulls.

## 🐛 Bug Fixes

- **Phantowl (and other Elite Redux customs) no longer freeze the battle.** When one of these Pokémon entered the field, the game tried to compute its cry using vanilla form-id math that doesn't apply to custom species, hit an undefined lookup, and **threw — locking the battle with no way out**. Custom Pokémon now resolve their cry safely (they're simply silent, as they have no cry audio) instead of crashing.
- **Wrong party-menu icon for wild Redux-form Pokémon.** A vanilla Pokémon that spawned in its **Redux form** (e.g. a wild **Spearow Redux**) showed a completely wrong icon in the party/team menu — sometimes a Mega Charizard. Its icon was being looked up as a `"<id>-redux"` frame that doesn't exist in the vanilla icon atlas, and the fallback landed on a stray frame. It now falls back to the correct base-form icon, so the party menu shows the right Pokémon. *(Affects the Legacy and Default themes alike.)*
