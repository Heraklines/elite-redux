# Elite Redux 0.0.3.6

## ☁️ Save System

- **Accounts + cloud saves.** You can now create an Elite Redux account (username + password) and your progress is stored in the cloud — so it survives clearing your browser cache and follows you to other devices/browsers.
- **One-time import of your existing save.** The first time you log in, if there's a save already on your device, the game offers to import it into your account (your local copy is kept — nothing is deleted).
- **Local-first, resilient sync.** Your save is written locally on every save, and synced to the cloud periodically. A temporary server hiccup no longer resets your run — it just retries later.
- **No more false "Save data could not be found" scare.** That message used to flash on a brand-new account a split second before your imported data loaded fine. It now only appears when there's genuinely nothing to load.
- **Username/password only.** Discord/Google login buttons are hidden (they aren't used by Elite Redux).

## 👻 Run History & Ghost Trainers

- **Your runs are now recorded.** Every finished run — win *or* loss, at any wave — is saved with your team, the opponent that ended it, the wave reached, and the outcome. (This also powers balancing.)
- **Ghost trainers come from real players.** The endgame ghost gauntlet now fields teams pulled from other players' actual runs, not just your own. The pool fills up as more people play.
- **Age-appropriate opponents.** A ghost can only appear at a wave the original run actually reached — a team that fell at wave 30 will never show up as a wave-180 boss. Ghosts still never repeat within a single run.
- **Existing progress seeds the pool.** When you first log in and import your local save, your stored run history is uploaded too, so the shared pool benefits from runs you played before accounts existed.

## ⚔️ Battles & Trainers

- **Rivals overhauled.** Rival battles now follow a clean six-stage progression (through a new Meteor Falls finale) with party sizes that scale stage by stage, and a run never fields the same rival team twice.
- **Angel's Wrath fully wired (Cascoon's signature).** Its enhanced moves now all work: Tackle locks the foe (Encore + Disable), Poison Sting badly poisons, Electroweb traps + crushes Speed and hits Ground super-effectively, Bug Bite drains HP — with the big power boosts. Cascoon (and Primal Cascoon) learn the full enhanced set.
- **Color Change refinement.** The pre-hit type-swap now compares against your Pokémon's current primary type, for more reliable resist/immunity swaps.
