# Elite Redux 0.0.3.6

## ☁️ Accounts & Cloud Saves
- Create an account (username + password) — progress is backed up to the cloud and survives cache wipes / new devices.
- First login offers to import your existing local save (your local copy is kept, never deleted).
- Local-first sync; a server hiccup no longer resets your run.
- Auto-updates onto new builds (saves untouched).

## 👻 Run History & Ghost Trainers
- Every run (win or loss) is recorded with your team and the opponent that ended it.
- Endgame ghost trainers are now pulled from other players' real runs (pool grows as people play).
- A run can only become a ghost at a wave it actually reached, and never repeats within a single run.

## ⚔️ Battles
- **Rivals reworked** — clean 6-stage progression (new Meteor Falls finale), scaling party sizes, never the same rival team twice in a run.
- **Angel's Wrath (Cascoon)** fully wired: Tackle Encores + Disables, Poison Sting badly poisons, Electroweb traps + crushes Speed + is super-effective vs Ground, Bug Bite drains, with power boosts.
- **Color Change** now swaps to a resisting/immune type *before* the hit.
- **Minion Control** no longer over-counts hits.
- **Psywave** scales off your stats (was near-zero).
- **Saber Slashes** hits twice.
- **Frostbite** is now exclusive with other statuses; every Freeze source inflicts Frostbite (no more "FRZ").
- No more **eternal weather**.
- **Calyrex** riders get their real As One ability, and the Pokédex no longer crashes on them.
- **Locust Swarm / Hivemind** direction fixed (Hivemind above ¼ HP).

## 🧬 Pokémon & Movepools
- TM/teachable movepools match ER — no illegal vanilla moves (e.g. Salazzle's Scald).
- A Redux form caught before its base form is now usable.

## ✨ Quality of Life
- **Innate filter** reworked for the 3-passive system: All / 2 / 1 / Can Unlock / None.
- **Buy max eggs** — one tap buys the most same-species eggs your candy allows.
- **Mass Unlock Innates** (Unlocks dropdown) — unlock every affordable innate across your whole box in one confirm, staggered so it never freezes.
- 🎁 One-time gift: **2 Legendary eggs** for everyone.
