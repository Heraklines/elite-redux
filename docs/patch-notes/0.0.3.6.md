# Elite Redux 0.0.3.6

## ☁️ Accounts & Cloud Saves
- Create an account (username + password) — progress backs up to the cloud and survives cache wipes / new devices.
- First login offers to import your existing local save (your local copy is kept, never deleted).
- Local-first sync; a server hiccup no longer resets your run.
- Auto-updates onto new builds automatically (saves untouched).

## 👻 Run History & Ghost Trainers
- Every run (win or loss) is recorded with your team and the opponent that ended it.
- Endgame ghost trainers are now pulled from other players' real runs (the pool grows as people play).
- A run can only become a ghost at a wave it actually reached, and never repeats within a single run.

## ⚔️ Battles & Abilities
- **Rivals reworked** — clean 6-stage progression (new Meteor Falls finale), party sizes scale by stage, and never the same rival team twice in a run.
- **Angel's Wrath (Cascoon)** fully wired: Tackle Encores + Disables, Poison Sting badly poisons, Electroweb traps + crushes Speed + is super-effective vs Ground, Bug Bite drains, all with power boosts.
- **Color Change** now swaps to a resisting/immune type *before* the hit lands.
- **Minion Control** no longer over-counts hits (full party = 6, lone holder = 1).
- **Psywave** scales off your stats now (was near-zero chip).
- **Saber Slashes** hits twice.
- **Calyrex** Ice/Shadow Rider get their real As One ability, and the Pokédex no longer crashes on them.
- **Frostbite** is now exclusive with other statuses, and every Freeze source inflicts Frostbite — no more "FRZ".
- No more **eternal weather** (Sunny day stuck at "Turns Left: 0").
- **Locust Swarm / Hivemind** direction fixed (Hivemind *above* ¼ HP).
- **Battle Bond** generalized — Darmanitan Redux Bond → Blunder now transforms (with correct sprites).

## 🎨 Sprites & Forms
- **Mega Goodra** and every **Redux-custom Mega** (plus their base forms) render real ER art instead of 404ing — fixes the battle freeze on those mons.
- ER-custom sprites with a baked-in background (e.g. **Redux Noibat**) are cleaned up to transparent.
- A middle-stage Redux mon no longer evolves into a **vanilla Mega** (e.g. Redux Kadabra → Mega Alakazam).
- A **Redux form caught before its base form** is now usable.

## 🧬 Movepools
- **TM/teachable movepools match ER** — no illegal vanilla moves (e.g. Salazzle's Scald).

## ✨ Quality of Life
- **Innate filter** reworked for the 3-passive system: All / 2 / 1 / Can Unlock / None.
- **Buy max eggs** — one tap buys the most same-species eggs your candy allows.
- **Mass Unlock Innates** (Unlocks dropdown) — unlock every affordable innate across your whole box in one confirm, staggered so it never freezes.
- **Elite/Hell field a different cast each run** (regular waves draw from the whole trainer pool, still wave-scaled).
- 🎁 One-time gift: **2 Legendary eggs** for everyone.
